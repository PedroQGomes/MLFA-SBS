# MIEI - 4º Ano - Sistemas Baseados em Similaridade

Trabalhos práticos sobre Machine Learning com aprendizagem supervisionada e não supervisionada.


## TP1

O primeiro trabalho pratico consiste na implementação de modelos de Machine Learning utilizando a plataforma
Knime e modelos baseados em árvores, como Árvores de Decisão ou Random Forests, para  prever acidentes na cidade de Braga e também para prever a influencia de atributos sociais e demográficos da performance de alunos de secundário.


Deste modo algumas das técnicas utilizadas foram as seguintes

* Forward e backward feature selection
* Cross Validation
* Decision tree
* Random forest 


## TP2 

O segundo trabalho pratico consiste na implementação de um Sistema de Recomendação de livros com interface gráfica. Para o desenvolvimento do sistema de recomendação foi usado python/knime e para a interface gráfica Asp.net core Web Pages.



Os tipos de sistemas de recomendação utilizados foram os seguintes

* Sistemas de Recomendação baseados em Conteúdo
* Sistemas de Recomendação de filtragem colaborativa


Deste modo as recomendações de livros implementadas, para cada utilizador, no trabalho pratico foram as seguintes


* Recomendações Baseadas em Conteúdos
* Recomendações Top World 
* Recomendações Top Country
* Recomendações Top Recents
* Recomendações Top Authors
